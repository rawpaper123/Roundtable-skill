# Roundtable Agent Roster Example

Use this as a planning aid. Do not treat roles as permanent.

| Agent | Dynamic expert angle for this task | Required? |
| --- | --- | --- |
| agent-a | Runtime/release reliability | yes |
| agent-b | Product trust and user impact | no |
| agent-c | Security/privacy review | no |
| agent-d | Adversarial scope reviewer | no |

## Rule

Assign expert angles per task. The same agent may cover different angles in a
future round.
