# Example: Generic Product Goal

```text
Open Roundtable Skill.

Project: <path>
Goal: improve the onboarding flow without changing the payment system.
Constraints:
- no database migration
- no dependency changes
- no production deploy unless explicitly requested

Assign dynamic expert angles:
- product trust
- frontend UX
- backend/API contract
- adversarial scope reviewer

If an expert has no actionable concern, they must reply no opinion.
Executor owns the final implementation, validation, Git, PR, and rollback.
```
